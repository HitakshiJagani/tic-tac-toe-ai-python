# 🎮 Tic-Tac-Toe AI (Python)

A command-line Tic-Tac-Toe game where you play against an AI powered by the Minimax algorithm. The AI always makes the optimal move, making it unbeatable.

## 🚀 Features
- Play against AI
- Intelligent moves using Minimax algorithm
- Detects win, loss, and draw
- Clean command-line interface

## 🧠 AI Logic
The AI uses the Minimax algorithm to evaluate all possible moves and choose the best one.

## 🛠️ Tech Stack
- Python
- Minimax Algorithm

## ▶️ How to Run
```bash
python tictactoe.py
